
# Fake Job Posting Detection System

## Overview
This project detects fraudulent job postings using Python, NLP, and Machine Learning.

## Technologies
- Python
- Pandas, NumPy
- NLP (TF-IDF)
- Scikit-learn
- Streamlit

## How to Run
1. Install requirements
   pip install pandas scikit-learn streamlit

2. Run Streamlit app
   streamlit run app.py

## Dataset
Fake Job Postings Dataset (Kaggle)

## Result
Achieved over 90% accuracy using Logistic Regression.
